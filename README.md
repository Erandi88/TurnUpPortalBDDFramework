# TurnUp Portal BDD Automation Framework

## 📌 Project Overview
This project is a BDD (Behaviour Driven Development) test automation framework for the TurnUp Portal using Reqnroll (SpecFlow), Selenium WebDriver, and C#.

## 🛠 Tech Stack
- C#
- Reqnroll (SpecFlow)
- Selenium WebDriver
- NUnit
- Visual Studio

## ✅ Features Covered
- Login functionality
- Time & Material module
- Create, Edit, and Delete operations

## 📂 Project Structure
- Features
- StepDefinitions
- Pages
- Utilities

## ▶️ How to Run
1. Open the project in Visual Studio
2. Build a solution
3. Run tests using Test Explorer

## 🎯 Purpose
This project demonstrates BDD automation skills and framework design for QA/Test Analyst roles.
